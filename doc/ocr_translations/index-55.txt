all settings of the display menu (Tabs, etc.), the insert mode, uppercase,

Due to an error in the assembler versions lower than V1.1, both

And finally, the format offers the advantage of being able to protect the
source text with a password or the serial number of the assembler. For
example, someone who has stolen your source text cannot do anything with
it unless they also stole your assembler or know your password. The
password is of course stored in coded form.

You can set the password with CONTROL-” before saving. The dialog box that
appears (see Figure 5.10 on page 54) allows you to define a password and
either protect the source text with a password or provide it with serial
number protection. It is also possible to switch off the protection here.
The serial number protection prohibits it to load the source code in a
different assembler than your serial number. A password may contain any
ASCII code (exceptions are 0, 27 and 255). The password has to be re-
entered each time before loading, which can then become quite annoying.
You can therefore also click the button in the middle instead of the
password button. It contains your serial number. Then the source text can
only be loaded on an assembler with this serial number — another reason
not to pass on the OMIKRON.assembler. If everyone has an assembler with
your serial number, this protection is pretty pointless.
