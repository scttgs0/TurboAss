TRANSLATED
==========


3. With ⇑ the SHIFT key is meant, for example, it is used with ⇑^E for
   SHIFT-CONTROL-E (ASCII search and replace).

Under the "Help" menu, in case of doubt, you can watch these key codes
again at any time (see chapter 5.5.5 on page 80). If there are still
three points behind an entry, it means that selecting this function
still follows a query (e.g. a dialog box, the FileSelector...).


The "Atarı" Menu

    {image}
    Figure 5.2: The "Atari" Menu

The first menu item ("OMIKRON.Assembler V1.7 ...") shows (almost) the
same dialog box (see Figure 5.3 on page 47), which you have already
seen when starting the assembler. It contains information about the
loaded version of the OMIKRON.Assembler as well as your serial number.
You can also call this menu item with CONTROL-I via keyboard.


    {image}
    Abbildung 5.3: Die Info-Box

Underneath you can view the list of people with "thanks ...", somehow to
promote the OMIKRON.Assembler contributed (see Figure 5.4 on page 48).
You can also call this dialog box with ALT-I.
