TRANSLATED
==========


This function key corresponds to the Info command (see Chapter 6.6.8 on page
160), but publishes the data in a dialog box and not on the text screen.

{image}

Figure 6.5: The memory report display

6.5.17 F9 — List PC

It is listed from the PC (like the "List PC" command). In contrast to
Disassemble, no hex part will be, but possibly existing symbols are issued.

6.5.18 SHIFT-F9 — Direct

"Direct"4 Demolishes the screen and output the switch-on message.

6.5.19 F10 — Toggle Screen

"Switch" switches back and forth between the screen of the Omikron.debugger
and the debugged program. If necessary, the resolution and the monitor are
also switched. If you do not use an automatic monitor switch, you must then
switch or insert by hand.
