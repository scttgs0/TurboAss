6.4.2 Die Tastaturbelegung

Die Tastaturbelegung entspricht soweit es möglich ist, der Tastaturbelegung
der Assemblers, so daß sie wahrscheinlich keine Probleme haben werden.
Der Editor verfügt über eine ganze Reihe von Tastaturkommandos (siehe
Tabelle 6.2 auf Seite 115).

UNDO: eignet sich sehr gut zur Wiederholung des letzten
Befehls.

CONTROL-M: Rettet die Cursorzeile in einem internen Buffer,
welcher mit mit den Einstellungen (siehe Kapi-
tel 6,8 auf Seite 168) gespeichert wird.

ALT-M: Holt die gerettete Zeile aus dem Buffer zurück.

CONTROL/ALT 1-9: bevor man ein Unterprogramm traced, kann man
die Bildschirmseite des Debuggers (mit Cursor-
position) retten, um später wieder auf der Bild-
schirmseite arbeiten zu können. Mit 1-9 sind nicht
die Zifferntasten des Zehnerblocks gemeint.

Zusätzlich zu diesen Tasten gibt es noch einige, die spezielle Funktionen
haben (siehe Tabelle 6.3 auf Seite 116).

ALT-<=,-=>: Der Debugger merkt sich die letzten 256 Befehle in
einem internen Buffer, dem sogenannten Cache. Er
rettet zu jedem Befehl den kompletten Register-
satz, und Sie haben mit diesen Tasten die Möglich-
keit, rückwärts durch die Befehlsliste zu gehen.

6.4.2 The keyboard layout

The keyboard layout corresponds as far as possible to the keyboard layout
of the assemblers, so you probably won't have any problems. The editor has
a whole range of key commands (see table 6.2 on page 115).

UNDO:               very useful for repeating the last command.

CONTROL-M:          Saves the cursor line in an internal buffer, which is
                    saved with the settings (see chapter 6.8 on page 168).

ALT-M:              Retrieves the saved line from the buffer.

CONTROL/ALT 1-9:    before tracing a subprogram, you can save the debugger
                    screen (with cursor position) in order to be able to
                    work on the screen again later. 1-9 does not mean the
                    number keys on the ten-block.

In addition to these keys, there are some that have special functions (see
Table 6.3 on page 116).

ALT-<=,-=>:         The debugger remembers the last 256 commands in an
                    internal buffer, the so-called cache. It saves the
                    complete set of registers for each command, and you
                    can use these keys to go backwards through the
                    command list.
