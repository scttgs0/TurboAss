TRANSLATED
==========


With "from the beginning" the cursor jumps back to the first error in the
source after the last error. Then there is no “last line” setting; if there
are no further errors in the source, the cursor jumps behind the last line
of the source.

The character that must follow the semicolon in order to be able to jump to
this line with ALT-S can be set with "Search character". So if you set "#"
and then write ";#" at the beginning of a line, this line will be jumped to
as soon as you press ALT-(SHIFT-)S (see chapter 5.5.5 on page 56).

Editor 2... (SHIFT-F7): In this box (see Figure 5.33 on page 77) you can set
various things. "Backup copy" means that a backup copy is created when saving


    {image}
    Abbildung 5.33: Die Dialogbox „Editor 2“


SRC files. To do this, the old source is first renamed as a BAK file, then the
new one is saved with the SRC extension. This means that nothing can happen if
the power fails while saving (then at least you still have the old source).

Under "Keyboard click" you can switch off the keyboard click, but not the sound
that sounds when symbols are used for the first time.

"Symbol names" allows differentiation between upper and lower case letters.
