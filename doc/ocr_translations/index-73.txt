* The text that is to replace the old one must also contain appropriate
asterisks (not question marks). For example, if we want to replace all
"RTS" with "RTE", we write "RTS" in the first line and ""RTE*" in the
second.

The following entry is "Jump to line..." (ALT-Z). When you call it up,
a dialog box appears (see Figure 5.28 on page 71), which allows you to
enter a line number or a symbol name. With a symbol name, the line in
which the symbol is defined is jumped to. With symbols, quantifiers are
possible at the end of the name!
