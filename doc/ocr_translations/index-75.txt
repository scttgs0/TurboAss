ORIGINAL
========


CONTROL-SHIFT-S sucht im Gegensatz zu CONTROL-S aufwärts.

Das Menü „Block“


    {image}
    Abbildung 5.29: Das Menü „Block“


Das „Block“-Menü enthält alle Kommandos, die zur Sourcetextbearbeitung
notwendig sind:

CONTROL-B: Markiert den Blockanfang (geht auch mit der Maus)
CONTROL-K: Markiert das Blockende (geht auch mit der Maus)
CONTROL-H: Löscht die Blockmarkierungen (Hide Block)
CONTROL-V: Verschiebt den Block an die Cursorposition
ALT-C: Kopiert den Block an die Cursorposition (Copy Block)
ALT-Y: Löscht den Block

Zudem kann man per Tastatur an die Blockgrenzen springen:
ALT-B: Sprung zum Blockanfang
ALT-K: Sprung zum Blockende

Ferner können Sie im „Block“-Menü einen Block speichern sowie einen Block
ausdrucken.

TRANSLATED
==========


In contrast to CONTROL-S, CONTROL-SHIFT-S searches upwards.

The "Block" menu


    {image}
    Figure 5.29: The "Block" menu


The "Block" menu contains all commands that are necessary for source text
processing:

CONTROL-B: Marks the beginning of the block (you can also use the mouse)
CONTROL-K: Marks the end of the block (you can also use the mouse)
CONTROL-H: Deletes the block markings (Hide Block)
CONTROL-V: Moves the block to the cursor position
ALT-C: Copies the block to the cursor position (Copy Block)
ALT-Y: Deletes the block

You can also jump to the block boundaries by keyboard:
ALT-B: Jump to the beginning of the block
ALT-K: Jump to the block end

You can also save a block and print a block in the "Block" menu.
