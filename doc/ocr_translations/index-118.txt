Table 6.3: Additional key commands

ALT-{left}, {right}             This allows you to leaf through the cache (corresponds to the arrows to the left of the register display)
ALT-CLR/HOME                    Displays the current register (corresponds to the & left of the register display)

CONTROL-P                       Sets the PC to the current address
CONTROL-G                       The program starts
CONTROL-B                       Sets a breakpoint (see page 112)

SHIFT-SHIFT                     The current program is interrupted and it will be returned to the debugger

CONTROL-HELP                    The debugger ends
CONTROL-ALT-DELETE              WarmStart the Debuggers + Reset
CONTROL-ALT-SHIFT               Cold start of the computer
right-DELETE                    Storage is deleted !!!
ALT-SHIFT-HELP                  The screen content is saved {3}.
