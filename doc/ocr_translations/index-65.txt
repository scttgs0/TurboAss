ORIGINAL
======== 


Eine Zurück-
wandlung in eine vorzeichenbehaftete Zahl ist durch erneutes Drücken von
RETURN möglich. Um nun ein Ergebnis auch nutzen zu können, gibt es drei
Buttons, die links von den Ergebnissen stehen. Mit ihnen kann das Ergeb-
nis an die Cursorposition übernommen werden (gemäß ihrer Zahlenbasis).
Ferner ist auch die Übernahme der gesamten Formel an die Cursorposi-
tion möglich, indem man auf den Button „Formel übernehmen“ klickt. Der
„ENDE“-Button oder die Taste UNDO bringen Sie ohne Übernahme einer
Formel in den Editor zurück. Es sei an dieser Stelle noch einmal daran erin-
nert, daß ein Doppelklick mit der rechten Maustaste auf eine Zahl, Formel
etc. den Taschenrechner aufruft, und, daß die Zahl, Formel o.ä. in densel-
ben übernommen wird. Noch eine nicht unwichtige Bemerkung am Rande:
Der Taschenrechner kennt keine Fehlermeldungen; wenn ein Fehler gefunden
wurde, ist das Ergebnis stets 0, d.h., 1/0+100 ist 0, da bei 1/0 ein Fehler
auftrat.

Der nächste Menüpunkt ist „Zeichentabelle...“ (CONTROL-+). Falls Sie ein
Zeichen eingeben wollen, das Sie nicht auf der Tastatur finden und des-
sen ASCII-Code Sie nicht auswendig wissen’, können Sie hiermit alle 255
möglichen Zeichen des STs auf einmal zu sehen bekommen (Siehe Abbil-
dung 5.18 auf Seite 63). Sie können dann mit einfachem Anklicken das ent-
sprechende Zeichen auswählen. Es wird dann an die Cursorposition über-
nommen. Falls Ihnen keines der Zeichen zusagt, können Sie kurz die rechte
Maustaste drücken, eine beliebige Taste der Tastatur oder mit der linken
Maustaste außerhalb der Dialogbox klicken. Darauf verschwindet die Dialog-
box wieder.

TRANSLATED
==========

It can be converted back to a signed number by pressing RETURN again. In order
to be able to use a result, there are three buttons to the left of the results.
They can be used to transfer the result to the cursor position (according to
their number base). It is also possible to transfer the entire formula to the
cursor position by clicking on the "Transfer formula" button. The "END" button
or the UNDO button takes you back to the editor without accepting a formula.
At this point it should be remembered that a double-click with the right mouse
button on a number, formula, etc. calls up the pocket calculator, and that the
number, formula, etc. is transferred to it. Another not unimportant side note:
The pocket calculator does not recognize any error messages; if an error is
found, the result is always 0, i.e. 1/0+100 is 0, since an error occurred at 1/0.

The next menu item is "Character Map..." (CONTROL-+). If you want to enter a
character that you can't find on the keyboard and whose ASCII code you don't
know by heart, you can use this to see all 255 possible characters of the ST at
once (See Figure 5.18 on page 63). You can then select the appropriate character
by simply clicking on it. It is then transferred to the cursor position.
If none of the characters suit you, you can briefly press the right mouse button,
click any key on the keyboard or use the left mouse button outside the dialog box.
The dialog box then disappears again.
